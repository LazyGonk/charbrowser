# CharBrowser Icons

This directory should contain the application icons in the following formats:

- `32x32.png` - 32x32 pixel PNG icon
- `128x128.png` - 128x128 pixel PNG icon
- `128x128@2x.png` - 256x256 pixel PNG icon (retina)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

For now, the application will use default icons. To add custom icons, generate or design icons and place them in this directory.
