# Support

## Getting Help

If you need help using or contributing to CharBrowser, use one of these channels:

1. Open a GitHub issue for bugs or feature requests.
2. Open a discussion thread (if Discussions are enabled) for general questions.
3. Review `README.md` and `CONTRIBUTING.md` for setup and workflow guidance.

## Before Opening an Issue

Please include:

- Your operating system
- App version or commit hash
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs or screenshots

## Security Issues

Do not report security issues in public trackers.

See `SECURITY.md` for private reporting instructions.

## Response Expectations

Maintainers will respond as capacity allows. Clear reproduction steps and logs help us resolve issues faster.
